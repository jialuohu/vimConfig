## Introduction
Just for personal vim configuration.

## Usage
```
cd ~/Downloads && \
git clone https://github.com/jialuohu/vimConfig.git && \
cd vimConfig && \
cp vimrc ~/.vimrc && \
mkdir -p ~/.vim/syntax && \
cp syntax/nginx.vim ~/.vim/syntax/ && \
mkdir -p ~/.vim/plugin && \
cp plugin/auto-pairs.vim ~/.vim/plugin && \
cd ~/Downloads && rm -rf vimConfig
```
